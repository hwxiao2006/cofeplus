# Figma 导入页面清单

使用方式（推荐）：
1. 在 Figma 安装并打开 `html.to.design` 插件。
2. 打开下表对应的 `.figma.html` 文件，复制全部代码。
3. 粘贴到插件的 HTML 输入框中导入。

| 页面 | 对应可粘贴代码文件 |
|---|---|
| 客户管理 | `figma-paste/customers.figma.html` |
| 设备入场 | `figma-paste/device-entry.figma.html` |
| 设备管理 | `figma-paste/devices.figma.html` |
| 点位管理 | `figma-paste/locations.figma.html` |
| 发货清单 | `figma-paste/materials-orders.figma.html` |
| 物料补充 | `figma-paste/materials-refill.figma.html` |
| 物料管理 | `figma-paste/materials.figma.html` |
| 菜单管理 | `figma-paste/menu.figma.html` |
| 订单管理 | `figma-paste/orders.figma.html` |
| 商品详情 | `figma-paste/product-detail.figma.html` |
| 商品管理 | `figma-paste/product-management.figma.html` |
