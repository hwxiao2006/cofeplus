3. 数据持久化

  • 排序结果存入 localStorage，key 为 categoryOrder
  • 存储分类 key 的有序数组，如 ["冷萃系列", "经典咖啡", "3D拉花"]

  • 新增分类自动追加到末尾
  • 删除的分类从排序数组中自动清理

4. 技术方案

  • 使用原生 HTML5 Drag and Drop API，无外部依赖
  • 在分类按钮上添加 draggable="true"
  • 监听 dragstart / dragover / dragend / drop 事件
  • 拖拽过程中动态计算插入位置，实时移动 DOM 元素
